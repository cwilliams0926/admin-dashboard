# admin-dashboard

A simple admin dashboard using CSS grid.

# How it's made

**Tech Used**: HTML and CSS

The HTML was written first, making sure to use semantic tags and putting thought into
the structure of the page. Grid was used to build the layout of sidebar, header, and main.
The styling was done one section at a time.

# Lessons learned

This put everything I've learned about Grid so far into practice. I still feel
like a complete novice but this helped me apply some of what I've learned about
the layout style.
